# CC Switch Next v3.16.5

## Highlights

- **Default Sonnet model updated to Claude Sonnet 5** — Anthropic, OpenRouter, and AWS Bedrock provider presets now default to the latest Claude Sonnet 5 model (1M context window, 128K max output tokens)
- **omp stability improvements** — Fixed several bugs related to omp (Oh My Pi) provider configuration, MCP synchronization, and test coverage gaps

## Provider Presets

- Anthropic direct API provider now defaults to `claude-sonnet-5`
- OpenRouter presets updated across all supported apps to use `anthropic/claude-sonnet-5`
- AWS Bedrock presets updated to use `global.anthropic.claude-sonnet-5`
- Merged PR [#6](https://github.com/benzhoupo/cc-switch-next/pull/6)

## Bug Fixes

- Fixed missing `omp` field in `McpApps` struct initializers across multiple test files
- Fixed omp directory configuration mock in `useDirectorySettings` test
- Removed stale `openclaw` field references in import/export sync tests
- Disabled Dependabot automatic dependency updates
- Enabled branch protection on `main` (`enforce_admins: true`)
